Display: Flex

Display: flex em CSS define que o elemento se tornará um container flexível, ou seja, um Flex Container. Com isso, todos os elementos filhos se tornam Flex Items e passam a ser posicionados de forma mais dinâmica e controlada.

Principais características:

  • Os itens ficam alinhados em uma única linha (por padrão).
  • É possível mudar o eixo principal (linha ou coluna).
  • Permite alinhamento e espaçamento fácil entre os itens.
  • Ideal para layouts responsivos e centralizações.

Propriedades:

flex-direction

É uma propriedade usada em um container flexível (ou seja, um elemento com display: flex).
para definir a direção principal em que os itens flexíveis serão organizados dentro do container.

display: flex;
flex-direction: row;
  • row (padrão): Os itens são dispostos em linha, da esquerda para a direita (padrão).
  • row-reverse: Os itens são dispostos em linha, mas da direita para a esquerda.
  • column: Os itens são dispostos em coluna, de cima para baixo.
  • column-reverse: Os itens são dispostos em coluna, mas de baixo para cima.
justify-content

A propriedade justify-content em CSS é usada para alinhar os itens ao longo do eixo principal dentro de um container flexível (display: flex).
Ou seja, ela controla como o espaço extra é distribuído entre os itens (ou ao redor deles), de acordo com a direção definida por flex-direction.

display: flex;
justify-content: center;
  • flex-start (padrão): Alinha os itens no início do eixo principal (padrão).
  • flex-end: Alinha os itens no final do eixo principal.
  • center: Centraliza os itens.
  • space-between: Distribui os itens com espaço igual entre eles (sem espaço nas bordas).
  • space-around: Distribui os itens com espaço igual ao redor de cada item (metade nas bordas).
  • space-evenly: Distribui os itens com espaço totalmente igual entre todos (incluindo bordas).
align-items

A propriedade align-items em CSS é usada para alinhar os itens ao longo do eixo transversal (ou eixo cruzado) dentro de um container flex (display: flex).
Enquanto justify-content alinha horizontalmente (no eixo principal), align-items alinha verticalmente (no eixo perpendicular) — a menos que o eixo principal tenha sido alterado com flex-direction.

display: flex;
align-items: center;
  • stretch (padrão): Os itens se esticam para preencher todo o eixo transversal.
  • flex-start: Alinha os itens ao início do eixo transversal.
  • flex-end: Alinha os itens ao final do eixo transversal.
  • center: Centraliza os itens no eixo transversal.
  • baseline: Alinha os itens pela linha de base do texto.
align-content

A propriedade align-content em CSS é usada para alinhar as linhas do conteúdo flexível ao longo do eixo transversal, somente quando há múltiplas linhas (ou seja, quando o flex container tem flex-wrap: wrap ou wrap-reverse).
Ela é parecida com align-items, mas enquanto align-items alinha os itens individuais, align-content alinha as linhas dos itens.

display: flex;
flex-wrap: wrap;
align-content: center;
flex-wrap

A propriedade flex-wrap em CSS define se os itens flexíveis devem ou não quebrar linha quando não houver espaço suficiente no container.
Por padrão, os itens em um container com display: flex ficam em uma única linha. Com flex-wrap, você pode permitir que eles “quebrem” para a próxima linha, como um wrap mesmo — tipo quando o texto pula para baixo ao final da linha.

display: flex;
flex-wrap: wrap;
  • nowrap (padrão): Todos os itens permanecem em uma única linha, mesmo que transbordem o container.
  • wrap: Os itens quebram para a próxima linha quando necessário (de cima para baixo).
  • wrap-reverse: Os itens também quebram, mas as linhas são empilhadas de baixo para cima (ordem reversa).
gap

A propriedade gap em CSS é usada para definir o espaçamento entre os itens dentro de containers com display: flex, grid ou inline-flex/grid.
Ela substitui a necessidade de usar margin-right, margin-bottom, etc., entre os itens — deixando o código mais limpo e fácil de manter.

display: flex;
gap: 16px;

Display: Grid

O display: grid em CSS é uma propriedade que transforma um elemento em um container de grid, permitindo organizar seus elementos filhos em linhas e colunas de forma precisa e controlada. É uma das formas mais poderosas de criar layouts responsivos e complexos com CSS.

Vantagens do Grid:

  • Controle total sobre linhas e colunas
  • Suporte a alinhamento vertical e horizontal
  • Criação de layouts responsivos com facilidade
  • Pode ser combinado com outras técnicas como Flexbox

Propriedades:

Template Columns

A propriedade grid-template-columns em CSS é usada para definir o número de colunas e o tamanho de cada uma dentro de um container com display: grid.

grid-template-columns: valor1 valor2 valor3 ...;
  • px: pixels
  • %: porcentagem
  • fr: fração do espaço disponível
  • auto: tamanho automático baseado no conteúdo
  • min-content

    • Define o tamanho mínimo necessário para que o conteúdo não quebre ou transborde.
    O elemento só vai ocupar o espaço que ele realmente precisa (por exemplo, o menor tamanho de uma palavra sem quebrar).

    grid-template-columns: min-content;

    se o conteúdo for uma palavra longa, ele só vai ter largura suficiente pra caber essa palavra sem quebrar.

  • max-content:

    • Define o tamanho máximo que o conteúdo pode ocupar sem quebrar.
    Ele estica até mostrar todo o conteúdo em uma linha, sem cortar nem dividir.

    grid-template-columns: max-content;

    Se tiver várias palavras, ele vai esticar a célula até mostrar tudo em uma linha só, sem quebra.

  • min-content() :

    • Define um intervalo de tamanho entre um mínimo e um máximo.

    grid-template-columns: minmax(200px, 1fr);

    “Essa coluna deve ter no mínimo 200px, mas pode crescer até ocupar 1fr do espaço, se sobrar.”
